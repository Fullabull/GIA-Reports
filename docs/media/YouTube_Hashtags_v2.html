Important Note: How People Actually Search on YouTube

Most people do not type hashtags when searching on YouTube.

They type normal words, like:

  • ambient space music
  • sci fi spoken word
  • cinematic soundscape
  • meditative music

YouTube then:

  • matches those words against titles, descriptions, transcripts
  • and only secondarily uses hashtags to help confirm the category

Hashtags mainly do two things:

  1. Create a clickable feed if someone taps #SciFiMusic
  2. Give YouTube a light hint about context

They are not the primary search method.

Rule:
Write titles and descriptions for how humans search.
Use hashtags only to reinforce the category.
